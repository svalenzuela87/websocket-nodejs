NODE JS DEVELOPMENT 


Pre-requisites:

-Browser (Chrome, IE, Opera, Mozilla)
-Node JS (https://nodejs.org/es/download/)


Running:

-Execute by command line the file ../server.js with Node. 
-Browse the file ../index.html .
-Control video played.


Resumen:

The server file will open a port 9090 in localhost, be sure that the port is not used by another app or SO.
